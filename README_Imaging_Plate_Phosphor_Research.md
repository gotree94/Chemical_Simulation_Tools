# X-ray CR Imaging Plate (IP) 형광물질 연구 정리

## 📌 개요

본 문서는 **X-ray Computed Radiography (CR) 시스템에서 사용되는 Imaging Plate(IP) 내 저장 형광물질(Storage Phosphor)** 에 대한 연구 자료를 정리한 것입니다.

CR 시스템은 X선을 저장했다가 레이저 자극 시 빛을 방출하는 **Photostimulated Luminescence (PSL)** 현상을 이용하여 영상을 획득합니다.

---

# 1️⃣ CR Imaging Plate의 기본 원리

## 🔬 Photostimulated Luminescence (PSL) 과정

1. X-ray 조사 → 전자 여기
2. 전자가 트랩(결함 준위)에 포획
3. 적색 레이저 조사
4. 포획된 전자 방출
5. Eu²⁺ 중심에서 청색광 방출
6. 광 검출 → 디지털 영상 생성

---

# 2️⃣ 대표적인 저장 형광물질

| 물질 | 특징 | 적용 |
|------|------|------|
| BaFBr:Eu²⁺ | 가장 전통적 CR 형광체 | 의료 CR |
| BaF(Br,I):Eu²⁺ | 해상도 개선형 | 고해상도 CR |
| CsBr:Eu²⁺ | Needle 구조, 고감도 | 최신 CR |

# 📎 요약

CR Imaging Plate는 주로 **Eu²⁺ 활성 할라이드 형광체**를 사용하며, 성능은 다음 요소에 의해 결정됩니다:

- 트랩 구조
- 결정 구조
- 도핑 농도
- 입자 크기 및 층 두께

형광체 설계는 곧 영상 품질과 직결됩니다.
